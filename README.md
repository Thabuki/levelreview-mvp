# LevelReview — MVP de Gestão Ágil de Projetos e Produtos

Este repositório contém a entrega do MVP desenvolvido para a Sprint de Gestão Ágil de Projetos e Produtos.

O produto escolhido foi o **LevelReview**, uma plataforma web simples para ajudar times de desenvolvimento de jogos a organizar feedbacks sobre fases, mapas e ambientes jogáveis em produção.

## Sobre o produto

Em projetos de jogos, feedbacks de level design muitas vezes ficam espalhados em chats, documentos, planilhas ou ferramentas genéricas de gestão. Isso pode causar perda de informação, retrabalho e dificuldade para entender quais levels precisam de mais atenção.

O **LevelReview** busca centralizar essas informações, permitindo que o time organize projetos, cadastre levels e acompanhe melhor o processo de revisão das fases do jogo.

## Objetivo do MVP

O objetivo do MVP é validar se uma ferramenta simples, focada em organização de projetos e levels, pode ajudar o time a reduzir perda de informação e preparar melhor o fluxo de revisão de level design.

Nesta primeira versão, o escopo foi mantido pequeno para validar a base do produto antes de avançar para funcionalidades mais completas, como feedbacks detalhados, dashboards, filtros e recursos visuais.

## Escopo da Sprint 1

A Sprint 1 foi planejada com foco no início do fluxo do produto:

- Cadastro de projeto
- Cadastro de level
- Lista de levels

Essas funcionalidades permitem que o usuário crie um projeto, cadastre levels dentro dele e visualize os levels existentes.
